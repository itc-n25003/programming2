## 2)TCPクライアントのサンプルコードを改造

#### 2で実行したサンプルコードのtarget_hostを変更し、別のWebページのHTMLを表示してみましょう。ただ、サンプルプログラムは暗号化されていないページしかうまくいかないので、httpでアクセス可能なホスト名を指定する必要があります。

### example.comやhttpbin.orgなどはhttpでアクセス可能なので、これらを指定してみましょう（時間があれば他のページも探してみましょう）。なお、送信するデータも少し変更する必要があります。client.sendの引数の文字列も変更しましょう。
```python
(venv3) $ cat tcp_client_0512.py 
import socket

target_host = "www.example.com"
target_port = 80

# ソケットオブジェクトの作成
client = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# サーバーへ接続
client.connect((target_host, target_port))

# データの送信
client.send(b"GET / HTTP/1.1\r\nHost: example.com\r\n\r\n")

# データの受信
response = client.recv(4096)

print(response.decode())
client.close()
````

## 3)UDPクライアントのサンプルコードを実行

#### 教科書p.11の「2.3UDPクライアント」から書かれているコードの解説を読み、サンプルコード（udp_client.py）を実行してみましょう。ただ、実行しても何も起こりません。tcpの時は受け手（サーバ）がいましたが、このサンプルコードにはいないからです。udpのサーバプログラムは以下のようになります。

import socket

host = "0.0.0.1"
port = 12345

sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.bind((host, port))

print(f"UDPサーバー起動中... {host}:{port}")
while True:
    data, addr = sock.recvfrom(1024)
    print(f"受信: {data} from {addr}")  # バイナリデータをそのまま表示
    if data == b"exit":
        print("終了コマンドを受信。サーバーを終了します。")
        break

sock.close()

## 4)UDPサーバプログラムを改造

#### サーバプログラムは、exitという文字列を受信すると終了するようになっていますが、他の文字列を受信すると別の動作をするようにしてみましょう。
例）helloを受信すると、worldを表示する

```python
mport socket

host = "127.0.0.1"
port = 55555

sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.bind((host, port))

print(f"UDPサーバー起動中... {host}:{port}")
while True:
    data, addr = sock.recvfrom(1024)
    print(f"受信: {data} from {addr}")  # バイナリデータをそのまま表示
    if data == b"exit":
        print("終了コマンドを受信。サーバーを終了します。")
        break
    elif data == b"hello":
        print("world")
        break

sock.close()
```
